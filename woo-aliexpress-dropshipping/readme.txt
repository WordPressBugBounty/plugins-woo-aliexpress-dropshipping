== Description ==

Sharkdropship for AliExpress, eBay, Amazon, Etsy and Temu is a comprehensive WordPress plugin that enables you to import products from multiple popular e-commerce platforms directly into your WooCommerce store.

**Key Features:**

* **Multi-Supplier Support**: Import products from AliExpress, eBay, Amazon, Etsy, and Temu
* **Chrome Extension Integration**: One-click product import with our Chrome extension
* **Product Management Dashboard**: View, edit, and manage all your imported products
* **Analytics & Reporting**: Track views, sales, and revenue for each product
* **Advanced Filtering**: Filter products by supplier, views, and other criteria
* **Responsive Design**: Works perfectly on desktop and mobile devices

**Supported Platforms:**
* AliExpress
* eBay
* Amazon
* Etsy
* Temu

**How It Works:**
1. Install the Sharkdropship Chrome extension
2. Browse products on any supported platform
3. Click the extension icon to import products
4. Manage and track performance in your WordPress admin

**Requirements:**
* WordPress 5.0 or higher
* WooCommerce 5.0 or higher
* PHP 7.4 or higher

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/sharkdropship-multisupplier/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install the Chrome extension from the Chrome Web Store
4. Start importing products from your favorite suppliers

== Frequently Asked Questions ==

= Do I need the Chrome extension? =

Yes, the Chrome extension is required for product import functionality. It allows you to import products with one click while browsing supported platforms.

= Which platforms are supported? =

The plugin supports AliExpress, eBay, Amazon, Etsy, and Temu.

= Is WooCommerce required? =

Yes, WooCommerce is required as this plugin integrates with WooCommerce for product management.

= Can I track product performance? =

Yes, the plugin provides detailed analytics including views, sales, and revenue tracking for each product.
